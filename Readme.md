# Hospital Management System (DBMS)

This project is a Hospital Management System developed using MySQL as part of a DBMS academic project.

## Features
- Normalized database design
- Primary and foreign key constraints
- JOIN queries
- Views
- Stored procedures
- Triggers

## Technologies Used
- MySQL
- SQL

## How to Run
1. Create a database in MySQL
2. Execute schema/create_tables.sql
3. Execute data/sample_data.sql
4. Run queries from queries folder
